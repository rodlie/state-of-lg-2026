# PixiEditor

## Slide 0

PixiEditor is a universal, node based 2D editor that was made to provide you with tools and features for all your 2D needs. Create beautiful sprites for your games, animations, edit images, create logos. All packed up in an intuitive and familiar interface.

## Slide 1

PixiEditor 2.0 released in the middle of 2025, it was the biggest milestone in its 7 year history. It evolved from a simple pixel-art editor into a fully featured,
node powered 2D editor. Among many other things, version 2.0 brought:

- A fully customizable Node Graph with prodecural artwork and animation capabilities,
- Animation timeline with frame-by-frame and procedural support,
- Home-grown universal 2D renderer capable of mixing vector and raster graphics,
- SVG interpreter built entirely from scratch,
- Dedicated toolsets for pixel-art, painting and vectors
- Ability to import .ttf + .otf files and edit them as vectors
- Custom Vulkan and OpenGL renderers

## Slide 2

The main goal for 2026 is to bring Node-based Brush Engine, Extension system and reusable Node Components alive.

Brush Engine is already available to test, it's based on the very same Node Graph, that is used for PixiEditor's documents. They claim, that it
is one of, if not the most advanced Brush Engine out there.
